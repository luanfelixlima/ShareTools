ShareTools

Compartilhamento inteligente de ferramentas entre condôminos

Sobre o Projeto

O ShareTools é uma plataforma desenvolvida para permitir que moradores de um condomínio compartilhem suas ferramentas de forma organizada, prática e segura.

Acessar o Sistema

Funcionalidades

  • Cadastro e login de usuários
  • Cadastro de ferramentas
  • Empréstimo e devolução de ferramentas
  • Dashboard com informações úteis
  • Controle de disponibilidade

Valores e Práticas

  • Sustentabilidade
  • Viabilidade Financeira
  • Conformidade com a LGPD
  • Inovação Social

Projeto acadêmico desenvolvido na Faculdade Engenheiro Salvador Arena - Linguagem de Programação II
